# React Example

This example targets React 19 with Vite 8 and consumes the local library package.
Use the "Connect to broker" button in the UI to execute the same
`await client.connect(connectionProperties)` flow shown in the root README.

## Run

```bash
cd ../..
npm install
cd examples/react-vite
npm run dev
```

## Build

```bash
npm run build
```
